What is Pheonix Client?

Pheonix Client is a client to read from a repository of archives and install them hassle free, lets take for example you have a repository of lots of zip files, using Pheonix Client you just input the repository file (You can generate them) and use our UI to navigate which ones you want to install, this is also really useful for downloading free games, We do not endorse piracy whatsoever and any repository's created to commit piracy have nothing to do with our program's main function.
